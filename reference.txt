Index page

Available status values:

data-status="new" - Shows a green "NEW" badge
data-status="beta" - Shows an orange "BETA" badge
data-status="soon" - Shows a gray "SOON" badge and disables the link
data-status="" or omitted - No badge shown

sample code

        <div class="operation-card" data-status="new">
            <span class="status-badge status-new">NEW</span>
            <h3>Compress PDF</h3>
            <p>Reduce file size while maintaining quality</p>
            <a href="/compress">Compress File</a>
        </div>

        <div class="operation-card" data-status="">

        </div>

            <a href="/split">Split File</a>
        </div>

            <p>Add password protection to your PDFs</p>
